String Methods

  1. charCodeAt() = Returns the Unicode of the character at the specified index
  2. localeCompare() = Compares two strings in the current locale
  3. repeat() = Returns a new string with a specified number of copies of an existing string

    • Click the button to display the extracted part of the string.

      Note: The repeat() method is not supported in IE 11 (and earlier versions).

  4. substring() Extracts the characters from a string, between two specified indices

    • Click the button to extract characters from the string.

  5. startsWith() Checks whether a string begins with specified characters
  6. valueOf() Returns the primitive value of a String object
  7. trim() Removes whitespace from both ends of a string
