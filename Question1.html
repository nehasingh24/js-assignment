The Various methods in console Function are

  1. console.log

    2. This console Function displays the content inside the console.This is a type of print statement which is most used by the programmer.

  2. console.error

    3. This Function displays the error in the program inside the console in the web.

  3. console.warn

    4. This Function is used to give away a warning also inside the console

  4. console.table

    5. This type of Function is used to display the elements in the the table format.

  5. console.time

    6. This Function keeps a track of time that how much time the program takes to execute the code.

  6. console.timeEnd

    7. This Function will tell the end time of the program and display. it is kept at the end of the program.
